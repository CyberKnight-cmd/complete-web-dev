# Version Control Fundamentals

## Goal
Master Version control fundamentals: repo, commit, branch, merge, PRs within Phase 1 — Web Foundations.

## Learning Objectives
- Explain Git workflow from first principles
- Apply version control concepts in code without reference
- Debug typical Git failure modes

## Coverage
- Clear definitions of repo, commit, branch, merge, and PRs
- Mental models for Git workflow and collaboration
- Minimal demo code examples
- Common pitfalls and conflict resolution

## Deliverables
- 1–2 page notes documenting version control concepts
- Runnable code snippet demonstrating Git operations
- Screenshots/GIFs of Git workflow and tools

## Acceptance Criteria
- Submit repo link with complete implementation
- README with setup/run instructions
- Include tests where applicable
- Demonstrate Git understanding through working examples
