# Basic Web Security Awareness

## Goal
Master Basic web security awareness: XSS/CSRF/CORS, secret handling within Phase 1 — Web Foundations.

## Learning Objectives
- Explain web security vulnerabilities from first principles
- Apply security concepts in code without reference
- Debug typical security failure modes

## Coverage
- Clear definitions of XSS, CSRF, CORS, and secret handling
- Mental models for web security threats and mitigations
- Minimal demo code examples
- Common pitfalls and security vulnerabilities

## Deliverables
- 1–2 page notes documenting security concepts
- Runnable code snippet demonstrating security implementations
- Screenshots/GIFs of security tools and testing

## Acceptance Criteria
- Submit repo link with complete implementation
- README with setup/run instructions
- Include tests where applicable
- Demonstrate security understanding through working examples
