## 28. Why CSS Is Separate from HTML

Key idea:

* You can **change the entire look of a website**
* Without touching the HTML structure

This separation:

* Makes maintenance easier
* Encourages clean design
* Allows multiple visual themes for the same content

**Mental Hook:**
