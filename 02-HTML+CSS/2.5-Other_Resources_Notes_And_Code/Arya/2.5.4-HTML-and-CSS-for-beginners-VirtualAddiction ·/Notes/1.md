## 4. Major Overhaul in the Late 1990s (HTML 4 Era)

### HTML 4.01 (1999)

* Last major update before HTML5
* Introduced **global collaboration**
* Aimed at **internationalization**

### Universal Character Set Adoption

* HTML adopted **Unicode (Universal Coded Character Set)**

#### Why this mattered:

* Allowed representation of:

  * Accents
  * Non-English scripts
  * Global languages

**Result:**

* Better typography
* More accurate search engine indexing
* Global accessibility

## 6. Birth of HTML5 (Why It Was Needed)

HTML5 introduced to:

* Standardize modern web needs
* Improve consistency
* Reduce external dependencies

### Collaboration Behind HTML5

* **W3C** (World Wide Web Consortium)
* **WHATWG** (Web Hypertext Application Technology Working Group)

They teamed up in **2006** to:

* Reduce reliance on plugins (Flash, Silverlight)
* Improve error handling
* Replace heavy scripting hacks with markup

## 7. Local Storage & Cookies (Big Shift)

### HTML5 Feature: Local Storage

* Web pages can store data **directly in the browser**
* Reduces dependency on **HTTP cookies**

#### Advantages:

* Faster content delivery
* Better security
* Offline functionality

### Example (HTML5 Local Storage ‚Äì Conceptual JS)

```javascript

let user = localStorage.getItem("username");
```
