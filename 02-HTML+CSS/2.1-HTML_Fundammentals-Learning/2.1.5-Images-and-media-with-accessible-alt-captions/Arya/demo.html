Decorative Images

[Border Decoration]

Poor Alt Text Examples (Avoid These)

  • alt="image" - Too generic
  • alt="picture of a dog" - Unnecessary "picture of"
  • alt="A brown and white dog sitting on green grass in a park with trees in the background on a sunny day" - Too verbose
  • Missing alt attribute entirely

Responsive Images

Using srcset for Different Screen Sizes

[Responsive Image - Resize browser to see effect] 
<img src="image-800.jpg"
     srcset="image-400.jpg 400w,
             image-800.jpg 800w,
             image-1200.jpg 1200w"
     sizes="(max-width: 600px) 400px,
            (max-width: 1000px) 800px,
            1200px"
     alt="Product showcase">

Picture Element for Art Direction

[Art-Directed Image - Different crops for different screens] 
<picture>
    <source media="(max-width: 600px)" srcset="mobile-crop.jpg">
    <source media="(max-width: 1200px)" srcset="tablet-crop.jpg">
    <img src="desktop-crop.jpg" alt="Team collaboration">
</picture>

Detailed Chart Description

This bar chart compares quarterly revenue between 2023 and 2024. The data shows:

  • Q1 2023: $2.1M, Q1 2024: $2.8M (33% increase)
  • Q2 2023: $2.3M, Q2 2024: $3.2M (39% increase)
  • Q3 2023: $2.7M, Q3 2024: $3.9M (44% increase)
  • Q4 2023: $3.1M, Q4 2024: $4.5M (45% increase)

The chart demonstrates consistent growth across all quarters, with the highest growth rate in Q4.

Accessibility Testing Checklist

Image Accessibility Checklist

  • ✅ All images have alt attributes
  • ✅ Alt text is descriptive and concise
  • ✅ Decorative images have empty alt text
  • ✅ Complex images have detailed descriptions
  • ✅ Images have appropriate file names
  • ✅ Responsive images work across devices

Media Accessibility Checklist

  • ✅ Videos have captions or subtitles
  • ✅ Audio content has transcripts
  • ✅ Media controls are keyboard accessible
  • ✅ Auto-playing media is avoided or muted
  • ✅ Fallback content provided for unsupported formats
