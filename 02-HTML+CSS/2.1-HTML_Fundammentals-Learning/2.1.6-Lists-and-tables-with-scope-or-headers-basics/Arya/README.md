# Lists & Tables Demo

## Setup Instructions

1. Open `demo.html` in any modern web browser
2. Test with screen readers to verify list and table navigation
3. Use accessibility tools to validate structure

## Key Features Demonstrated

### List Types
- **Unordered lists**: Non-sequential items
- **Ordered lists**: Sequential steps or rankings
- **Description lists**: Term-definition pairs
- **Nested lists**: Hierarchical content organization

### Table Accessibility
- **Caption**: Provides table context
- **Scope attributes**: Define header relationships
- **Headers attributes**: Complex table navigation
- **Semantic structure**: thead, tbody, tfoot

## Testing Procedures

### Screen Reader Testing
- Navigate lists with L key (NVDA)
- Navigate tables with Ctrl+Alt+arrow keys
- Verify headers are announced with data cells

### Expected Results
- Lists are navigable as semantic structures
- Table headers provide context for data cells
- Complex relationships are clearly defined
- All content is accessible via keyboard
