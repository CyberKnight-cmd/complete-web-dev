Internationalization: Language, Direction & Entities Demo

This page demonstrates HTML internationalization features including language declaration, text direction, and character entities.

Language Declaration Examples

Multiple Languages

English: Hello, welcome to our website!

Spanish: ¡Hola, bienvenido a nuestro sitio web!

French: Bonjour, bienvenue sur notre site web!

German: Hallo, willkommen auf unserer Website!

Japanese: こんにちは、私たちのウェブサイトへようこそ！

Arabic: مرحبا، أهلا بك في موقعنا!

Code Example:

<p lang="es">¡Hola, bienvenido!</p>

Text Direction Examples

Right-to-Left Text

Default LTR: This text flows from left to right.

RTL Arabic: هذا النص يتدفق من اليمين إلى اليسار

RTL Hebrew: טקסט זה זורם מימין לשמאל

Bidirectional Text

Mixed: Hello مرحبا World שלום!

English with النص العربي embedded.

Code Example:

<p dir="rtl" lang="ar">النص العربي</p>

HTML Character Entities

Reserved Characters

< (less than) = &lt;
> (greater than) = &gt;
& (ampersand) = &amp;
" (quote) = &quot;
' (apostrophe) = &apos;

Accented Characters

á = &aacute;   é = &eacute;   í = &iacute;
ó = &oacute;   ú = &uacute;   ñ = &ntilde;
ç = &ccedil;   ü = &uuml;   ö = &ouml;

Symbols and Currency

© = &copy;   ® = &reg;   ™ = &trade;
€ = &euro;   £ = &pound;   ¥ = &yen;
§ = &sect;   ¶ = &para;   • = &bull;

Mathematical Symbols

∞ = &infin;   ≠ = &ne;   ≤ = &le;
≥ = &ge;   ± = &plusmn;   × = &times;
÷ = &divide;   √ = &radic;   ∑ = &sum;

Numeric Character References

Decimal and Hexadecimal

© = &#169; (decimal) = &#xA9; (hex)
€ = &#8364; (decimal) = &#x20AC; (hex)
™ = &#8482; (decimal) = &#x2122; (hex)
♥ = &#9829; (decimal) = &#x2665; (hex)

Accessibility Considerations

Screen Reader Benefits

  • ✅ lang attributes enable proper pronunciation
  • ✅ dir attributes control reading direction
  • ✅ Character entities ensure proper display
  • ✅ UTF-8 encoding supports all languages

Testing Checklist

  • □ Document language declared in html tag
  • □ Language changes marked with lang attribute
  • □ Text direction set for RTL languages
  • □ Special characters properly encoded
  • □ UTF-8 charset declared in head
