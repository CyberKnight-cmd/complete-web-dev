# Internationalization Demo

## Setup Instructions

1. Open `demo.html` in any modern web browser
2. Test with screen readers to hear pronunciation differences
3. View source to see language and direction attributes
4. Test character entity rendering

## Key Features Demonstrated

### Language Declaration
- Document-level language setting
- Section-specific language changes
- Regional language variants
- Proper pronunciation for screen readers

### Text Direction
- Left-to-right (LTR) default
- Right-to-left (RTL) for Arabic/Hebrew
- Bidirectional text handling
- Auto-direction detection

### Character Entities
- Reserved HTML characters
- Accented characters
- Symbols and currency
- Numeric character references

## Testing Procedures

### Screen Reader Testing
- Enable screen reader
- Navigate through different language sections
- Verify pronunciation changes with lang attributes
- Test RTL text navigation

### Expected Results
- Proper pronunciation for each language
- Correct text direction rendering
- All special characters display correctly
- No encoding issues with international text
