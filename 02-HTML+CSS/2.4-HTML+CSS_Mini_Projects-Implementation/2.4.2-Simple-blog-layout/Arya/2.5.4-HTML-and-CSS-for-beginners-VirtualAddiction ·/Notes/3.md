## 40. What Is Hypertext?

**Hypertext** =

* Text displayed on a screen
* That contains **links to other documents**

### Key Idea

* The web is a massive network of:

  * Billions of documents
  * Connected by **hyperlinks**

Example:

```html

```

This is hypertext in action.

## 50. Opening and Closing Tags

* Tags are wrapped in **angle brackets** `< >`
* Opening tag:

```html

```

* Closing tag:

```html

```

### Key Rule

* Closing tags include a **forward slash**
* Tags must be **properly nested**
