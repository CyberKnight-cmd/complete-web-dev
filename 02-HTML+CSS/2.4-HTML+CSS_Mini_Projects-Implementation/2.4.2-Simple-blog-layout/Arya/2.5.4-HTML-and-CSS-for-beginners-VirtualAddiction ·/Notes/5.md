## 111. Demonstrating a Common Error (Unclosed Tag)

Example mistake:

```html

```

What happens:

* Paragraph appears bold
* Browser thinks `<p>` is inside `<h3>`

Why?

* `<h3>` was **never closed**

---

## 112. Fixing the Error

Correct version:

```html

<p>Eels, fish, and sharks.</p>
```

Result:

* Heading and paragraph render correctly
