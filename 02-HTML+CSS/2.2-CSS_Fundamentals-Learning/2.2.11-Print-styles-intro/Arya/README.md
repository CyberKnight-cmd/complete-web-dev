# Print Styles Demo

## Setup Instructions
1. Open `demo.html` in any modern web browser
2. Use Ctrl+P (Cmd+P on Mac) to open print preview
3. Compare screen vs print layouts

## Key Features Demonstrated
- **Print media queries**: Different styles for print
- **Content control**: Hide/show elements for print
- **Typography optimization**: Serif fonts and point sizes
- **Page breaks**: Control page flow

## Testing Procedure
1. View page normally in browser
2. Open print preview to see print styles
3. Notice sidebar disappears
4. Observe typography changes
5. See print-only content appears

## Expected Results
- Screen version shows colorful layout with sidebar
- Print version shows clean, black text layout
- Unnecessary elements hidden in print
- Typography optimized for paper reading
