# Borders, Radius & Shadows Demo

## Setup Instructions
1. Open `demo.html` in any modern web browser
2. Observe different border styles and effects
3. Notice how radius and shadows create visual depth

## Key Features Demonstrated
- **Border styles**: solid, dashed, dotted
- **Border radius**: rounded corners, circles, complex shapes
- **Box shadows**: basic, inset, multiple shadows
- **Combined effects**: Modern card design

## Expected Results
- Different border styles create distinct visual effects
- Border radius softens sharp corners
- Shadows add depth and visual hierarchy
