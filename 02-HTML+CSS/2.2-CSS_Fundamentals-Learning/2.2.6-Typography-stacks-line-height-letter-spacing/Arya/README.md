# Typography Demo

## Setup Instructions
1. Open `demo.html` in any modern web browser
2. Compare different font families and their rendering
3. Observe line-height effects on readability
4. Notice letter-spacing impact on text appearance

## Key Features Demonstrated
- **Font stacks**: serif, sans-serif, monospace families
- **Line height**: tight (1.2), normal (1.5), loose (2.0)
- **Letter spacing**: condensed, normal, expanded variations

## Expected Results
- Different font families show distinct character styles
- Line height affects text readability and vertical rhythm
- Letter spacing changes text density and appearance
