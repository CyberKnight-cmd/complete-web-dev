# CSS Debugging Demo

## Setup Instructions
1. Open `demo.html` in any modern web browser
2. Open DevTools (F12)
3. Practice debugging the intentional CSS issues

## Debugging Exercises
1. **Specificity Conflict**: Inspect the colored text to see which style wins
2. **Box Model Issue**: Check why the box isn't the expected size
3. **Positioning Problem**: Examine the positioned element's values
4. **Visibility Issue**: Find and fix the nearly invisible element

## DevTools Features to Use
- Elements panel for DOM inspection
- Styles pane for CSS rules
- Computed tab for final values
- Box model viewer for sizing

## Expected Learning
- How to identify CSS conflicts
- Using DevTools to debug layout issues
- Understanding computed vs declared values
- Resolving common CSS problems
