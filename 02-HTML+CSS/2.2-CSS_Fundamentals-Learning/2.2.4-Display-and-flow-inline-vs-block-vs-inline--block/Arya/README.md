# Display & Flow Demo

## Setup Instructions

1. Open `demo.html` in any modern web browser
2. Observe how different display values affect layout
3. Resize browser to see flow behavior
4. Use DevTools to inspect computed display values

## Key Features Demonstrated

### Display Values
- **Block**: Full width, stacks vertically
- **Inline**: Flows with text, no width/height
- **Inline-block**: Flows horizontally, accepts dimensions

### Document Flow
- Normal flow behavior
- How elements position relative to each other
- Text wrapping with inline elements

## Expected Results
- Block elements take full width and stack
- Inline elements flow with text content
- Inline-block elements flow but can be sized
- Navigation demonstrates practical inline-block usage
