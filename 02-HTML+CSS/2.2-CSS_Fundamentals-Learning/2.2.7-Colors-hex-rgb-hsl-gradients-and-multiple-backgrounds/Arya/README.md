# Colors & Gradients Demo

## Setup Instructions
1. Open `demo.html` in any modern web browser
2. Observe different color format renderings
3. Compare gradient effects and combinations

## Key Features Demonstrated
- **Color formats**: Hex, RGB, HSL, RGBA
- **Gradients**: Linear and radial gradients
- **Multiple backgrounds**: Layered background effects

## Expected Results
- Different color formats produce same visual results
- Gradients create smooth color transitions
- Multiple backgrounds layer properly
