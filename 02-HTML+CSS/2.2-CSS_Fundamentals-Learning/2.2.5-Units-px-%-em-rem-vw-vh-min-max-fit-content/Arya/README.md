# CSS Units Demo

## Setup Instructions

1. Open `demo.html` in any modern web browser
2. Resize browser window to see responsive behavior
3. Use DevTools to inspect computed values
4. Test on different screen sizes

## Key Features Demonstrated

### Unit Types
- **Absolute**: px (fixed size)
- **Relative**: %, em, rem (contextual sizing)
- **Viewport**: vw, vh (viewport-relative)
- **Functions**: clamp(), min(), max()

### Responsive Behavior
- Percentage widths adapt to container
- Viewport units scale with browser size
- Clamp() provides fluid typography
- Min() creates flexible constraints

## Testing Procedures
- Resize browser to see viewport units change
- Compare em vs rem scaling behavior
- Test responsive text with clamp()
- Verify percentage calculations

## Expected Results
- Fixed px elements stay same size
- Percentage elements scale with container
- Viewport units change with browser size
- Modern functions provide flexible sizing
