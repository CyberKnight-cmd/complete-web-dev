# Tooling Landscape

## Goal
Master Tooling landscape: editors, terminals, package managers, devtools within Phase 1 — Web Foundations.

## Learning Objectives
- Explain development tooling from first principles
- Apply tooling concepts in code without reference
- Debug typical tooling failure modes

## Coverage
- Clear definitions of editors, terminals, package managers, and devtools
- Mental models for development workflow and toolchain
- Minimal demo code examples
- Common pitfalls and configuration issues

## Deliverables
- 1–2 page notes documenting tooling concepts
- Runnable code snippet demonstrating tool usage
- Screenshots/GIFs of development tools and workflow

## Acceptance Criteria
- Submit repo link with complete implementation
- README with setup/run instructions
- Include tests where applicable
- Demonstrate tooling understanding through working examples
